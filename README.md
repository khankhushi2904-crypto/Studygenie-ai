# StudyGenie AI

## Overview
StudyGenie AI is an AI-powered study assistant that helps students learn efficiently from PDF notes and study materials. Users can upload PDFs and generate summaries, important questions, flashcards, MCQs, and ask questions directly from the uploaded content.

## Features

- PDF Upload
- PDF Text Extraction
- AI-Powered Summary Generation
- Important Question Generation
- Flashcard Generation
- MCQ Generation
- Ask AI from PDF Content
- Download Summary Feature

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- PyPDF
- Python Dotenv

## How to Run

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Create a .env file and add your Gemini API key

GOOGLE_API_KEY=your_api_key_here

4. Run the application

streamlit run app.py
