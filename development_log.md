# Development Log

## Project Name

StudyGenie AI

## AI Coding Assistant Used

ChatGPT and Codex

## AI Model Used

Google Gemini 2.5 Flash

## Problem Statement

Students often spend a lot of time reading lengthy PDF notes and study materials. The objective of this project was to build an AI-powered study assistant that can summarize notes and generate useful study resources automatically.

## Important Prompts Given

### Summary Generation

Summarize the following PDF in simple points.

### Important Questions

Generate 10 important exam questions from the following PDF.

### Flashcards

Generate flashcards from the following PDF with Question and Answer format.

### MCQ Generation

Generate 10 MCQs from the following PDF with four options and the correct answer.

### Ask AI

Answer based only on the PDF content.
